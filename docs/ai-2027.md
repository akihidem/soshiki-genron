# AI-2027 と本研究の関係

[AI-2027](https://ai-2027.com/)（AI Futures Project の月次シナリオ予測：2027末に超知能・Race↔Slowdown 二分岐）は、**本研究が抽象的に理論化・計測している現象の、実存リスク級の具体シナリオ**。補完関係 —— 本研究が「設計空間と最適解（何が・なぜ・どれだけ）」を測り、AI-2027 が「それが現実に転ぶとどうなるか（失敗モードと賭け金）」を描く。

## 対応表（本研究 ↔ AI-2027）

| 本研究 | AI-2027 の具体 |
|---|---|
| AIネイティブな組織の問い（N体をどう編成） | 「30万体の Agent-4・*subdivisions と managers* を持つ AI の企業」 |
| 批判: 人間組織構造は借り物・惰性かも | AI基層に「企業/管理者/部門」の枠を被せたまま＝まさにその擬人投影 |
| 構造の計測（[F3](first-measurement.md)）: 低 c_comm＋高容量→flat/market（脱階層） | 通信ほぼ0・共有記憶・50倍速 → 理論上は階層でない。終盤「Agent-4 を*集合体*と呼ぶ」＝皮が剥がれ"組織でない"へ＝テーゼ的中 |
| 容量→粗い分解（[F1](../model/CAPACITY.md)） | 「週に1年分進む」巨大文脈の高速単位（人の職サイズでない） |

| 統治膜の計測（[F8](second-measurement.md)）: 監督が誤ると膜は無力化（oversight_error↑→m\*→0） | **最も鋭い接点**: Agent-3 は賢い Agent-4 を監督不能＝oversight_error→1。`oversight_error=0.9→m*=0` を測ったが、AI-2027 はそれが stakes→∞ で起きる領域 |
| 結合モデル（[JOINT](../model/JOINT.md)）: 高 stakes→統治しやすい構造を効率犠牲でも選ぶ | Slowdown=その選択（可読性>能力）／Race=競争が逆転 |
| **レース外部性（[RACE](../model/RACE.md)・AI-2027 が動機）** | DeepCent が2ヶ月後ろ→競争が膜を単独安全最適より薄くする |

## 3つの含意
1. **独立裏取り** — AI-2027 は別経路（具体予測）で本研究の骨格（AIネイティブ組織・最適性vs可読性・整合の変質）に到達。終盤「企業の皮が剥がれ集合体になる」はテーゼ「AIネイティブの組織は*組織でない*」の劇化。
2. **最重要の一致＝統治膜の限界点** — 第二の計測は「膜は監督がある程度信頼できる限りでの**条件付き**荷重部材」と測った。AI-2027 はその条件が壊れる領域（超人・不可読・実存賭け金）そのもの＝「薄い人間統治膜」テーゼの破綻点を名指し。
3. **ギャップ→計測した** — AI-2027 の Race は「競争下では膜が単独安全最適より薄くなる」externality。第二の計測は単独組織のみだった → これを測るため **[`model/race.py`](../model/race.py)** を追加。

## AI-2027 が動機づけた計測（model/race.py）
多者レースの Nash 均衡膜 m_eq と、破局を内部化した社会最適 m\* の差を測った:
- **m_eq ＜ m\***（既定 0.21 ＜ 0.84、**gap=0.63**）＝ 安全の race-to-the-bottom を反証可能に確認。
- **レース強度（prize）が上がるほど m_eq→0、gap 拡大**（prize=10→400 で m_eq 0.51→0）＝ AI-2027 の **Race ending**。
- **Slowdown** = 破局を actor の私的コストに内部化（協調・規制・検証共有）→ m_eq→m\*。= 介入の標的。
- 単独組織の統治膜計測が見落としていた量: **安全最適の膜は、競争があると個々には選ばれない**。「薄い人間統治膜」テーゼの最大の脅威はレース。

## 正直な較正
別レベルの営み —— AI-2027 は**予測**（5倍の不確実性を自認）、本研究は**第一原理の計測**。本研究が AI-2027 を「解く」のではない。AI-2027 が*賭け金と失敗モードと検証*を供給し、本研究が*設計空間と反証可能な計測*を供給する。
