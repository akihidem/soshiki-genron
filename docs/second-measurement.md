# 第二の計測 — 統治膜の最適な厚み（最適性 vs 可読性）

> [`foundations.md`](foundations.md) §5 の中心的緊張を測れる形に。生数値は [`../model/GOVERNANCE.md`](../model/GOVERNANCE.md)（再生成）。本ドキュメントは durable な解釈。

## 1. 問いを計測可能にする
- **概念（F8 / §5）**: AIネイティブに最適化された機構は人間に読みにくい。人間が主権・説明責任を保つには「人間可読な統治膜」を被せる必要がある。膜は効率には死荷重だが、人間が高リスクの誤りを捕まえる窓でもある。
- **測れる形**: 膜の厚み m∈[0,1]（AI機構の判断のうち人間可読・監査・承認を通す割合）に**最適値 m\* はあるか、何が決めるか**。

## 2. モデル（[`../model/governance.py`](../model/governance.py)）
```
総損失(m) = c_mem·m·N            （膜は判断ごとに効率税・通信非依存）
          + p_bad·N·stakes·e^(−β·m)   （人間監督が高リスク誤りを飽和的に捕捉）
```
効率コストは m に線形、捕捉便益は飽和（限界逓減）→ 内点最適が出る:
`m\* = clip( (1/β)·ln(β·p_bad·stakes/c_mem), 0, 1 )`。

## 3. 計測結果
- **内点最適が存在する**（既定 m\*≈0.73）。膜ゼロ（純効率）でも全面膜でもなく、**部分的な膜**が最小損失。
- **stakes がしきい値を決める**: stakes < **1.11** → m\*=0（膜は割に合わない＝純効率最適）。stakes > **22.3** → m\*=1（全面膜）。その間は薄い膜が最適で、stakes とともに対数的に厚くなる。
- **感度（監督の到達度 β）**: 監督が効くほど薄い膜で足り、低 stakes でも膜が割に合う。監督が無力なら膜は死荷重。
- **追補（監督の不確かさ oversight_error）**: 監督自身が誤ると捕捉に上限がつき、m\* は下がる（0→0.3→0.6→0.9 で m\*=0.73→0.61→0.43→**0**）。**「膜が荷重部材」は監督がある程度信頼できる限りでの条件付き主張**。
- **追補2（precision・実測由来 B）**: 実証（[`oversight-pilot.md`](oversight-pilot.md)）で「**強い監督ほど clean を過剰flag**」が再現（FP haiku 0 → opus 0.33）。これを over-flag コスト項として入れると、率↑で最適膜は薄くなり、**実測の opus 率 0.33 では m\*=0（膜が消える）**。→ 強い監督ほど厚い膜が良い、とは限らない。膜のコストは効率税だけでなく**誤検出ノイズ**も含む。

→ これは §5「膜が厚すぎれば擬人的負債、薄すぎれば主権喪失。最適は**必要十分に薄い膜**」を、**stakes と監督効率が決めるしきい値**として測ったもの。

## 4. 反証手段
- 任意の stakes>0 で m\*=0 のまま（膜が決して割に合わない）→ 「統治は荷重部材」は本モデル下で**偽**。
  実測ではしきい値（1.11）を超えると m\*>0 になり、**偽にならなかった** → §5 を条件付きで支持。
- m\* が stakes に依存しない → 「stakes が膜の厚みを決める」は偽。

## 5. 位置づけ
- foundations §5 の「最適性 vs 可読性」を、**定性の主張から定量のしきい値**へ進めた。膜は*常に*要るのでも*要らない*のでもなく、**stakes が高いほど厚い膜が最適**。
- 第一の計測（[`first-measurement.md`](first-measurement.md)）と合わせ、テーゼ「AIネイティブな機構＋薄い人間統治膜」の**両半分が測れる対象**になった: 機構の効率（通信コスト→構造）と、膜の最適厚み（stakes→m\*）。

## 6. 妥当性の限界
- 効率コスト線形・捕捉便益飽和は*仮定*。係数（特に β＝人間監督が確信的誤りをどれだけ捕まえるか）は ② か実データで測るべき経験量。結論は質的（内点最適の存在・stakes 依存）＋感度として読む。
- 「誤り」を高リスク one-shot で単純化。実際は誤りの分布・検出遅延・人間自身の誤判定がある（→ 次の精緻化）。

## 7. 次の計測
1. ~~人間の誤判定~~ **計測済み**（`oversight_error` 軸を追加・上記「追補」）: 監督が誤るほど m\* は低下、≈0.9 で膜は割に合わない。
2. 第一・第二の計測を**結合**: 構造（通信コスト）と膜（stakes）を同一空間で最適化（→ [`../model/design_map.py`] が第一近似。分離可能性の反証が残課題）。
3. β・oversight_error を**実測**する道（②の人間-in-the-loop 実験 or 既存データ）。
